# FOA Intelligence Monitor

A static MVP for an AI-powered real-time monitoring and intelligence alert platform, built from the provided PDF brief.

## Run

Open `index.html` in a browser. No install step is required.

## Included

- Live intelligence dashboard with priority filtering and search
- AI-style alert summaries with original text and English translation
- Multi-platform source health monitoring
- Searchable archive with CSV export
- Trend analytics and generated daily brief
- Simulated real-time detection flow

## Files

- `index.html` - app structure
- `styles.css` - responsive dashboard styling
- `app.js` - app data, rendering, filters, simulation, export, and reports
